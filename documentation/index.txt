====== Synconix ======
A Linux Audio Redistribution

Synconix is an linux based distribution designed for ~~crime~~ audio production.

 Nodes (soundbots) are configured from a management console using Ansible.

Download the Live ISO to get started...
	[ ] make iso 

The desktop environmnet uses i3 window manager.
